# 🎯 AI彩票预测分析系统

一个基于人工智能的彩票预测分析工具，支持双色球和大乐透的历史数据分析与智能预测。

## 📋 项目简介

本项目是一个使用PyQt5开发的桌面应用程序，集成了AI大语言模型（云雾AI）进行彩票号码预测分析。系统通过分析历史开奖数据，结合统计学和概率学原理，为用户提供专业的号码预测建议。

## ✨ 主要功能

### 🎲 支持的彩票类型
- **双色球**：红球01-33，蓝球01-16
- **大乐透**：前区01-35，后区01-12

### 🔍 核心功能
1. **历史数据获取**：自动从官方API获取最新开奖数据
2. **智能预测分析**：基于AI模型进行号码预测
3. **二次预测优化**：对首次预测结果进行频率分析和重新组合
4. **数据缓存管理**：本地缓存历史数据，提高查询效率
5. **多模型支持**：支持多种AI模型进行预测分析

### 📊 分析特性
- 冷热号分析
- 遗漏值分析
- 奇偶比例分析
- 大小比例分析
- 连号、重号、跨度分析
- 历史走势规律总结

## 🎮 使用指南

### 1. 启动程序
运行主程序后，会显示主界面，包含控制面板和结果显示区域。

### 2. 选择彩票类型
- 在"彩票类型"下拉框中选择"双色球"或"大乐透"

### 3. 选择期数
- 系统会自动加载最新期数
- 可以手动选择历史期数进行分析

### 4. 选择AI模型
- 从下拉列表中选择合适的AI模型
- 不同模型可能有不同的预测效果

### 5. 获取历史数据
- 点击"获取历史数据"按钮
- 系统会自动从官方API获取数据
- 支持强制刷新功能

### 6. 开始预测
- 点击"🎲 开始预测"按钮
- 系统会结合历史数据进行AI分析
- 预测结果会显示在结果面板中

### 7. 二次预测
- 基于首次预测结果进行频率分析
- 重新组合号码，提高中奖概率
- 结果会显示在"二次预测"标签页

## 🐛 故障排除

### 常见问题

1. **程序无法启动**
   - 检查Python版本是否为3.7+
   - 确认PyQt5已正确安装
   - 检查虚拟环境是否激活

2. **无法获取历史数据**
   - 检查网络连接
   - 确认API服务是否正常
   - 尝试使用强制刷新功能

3. **预测失败**
   - 检查AI API密钥是否有效
   - 确认网络连接稳定
   - 检查选择的AI模型是否可用

4. **界面显示异常**
   - 检查系统分辨率设置
   - 确认PyQt5版本兼容性
